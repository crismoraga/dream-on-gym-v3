## Project Description

DREAM-ON-GYM-V2: Deep Reinforcement Learning framework for Elastic Optical Networks (EON) using Flex-Net-Sim simulator. Extended with 5 reward function implementations including novel Spectral-Entropy Adaptive Reward.



## Architecture

Modular architecture: simNetPy (network simulation), gym_basic (RL environment), reward_functions (custom reward engineering). Supports multi-band EON (L, C, S, E, O) with slot-level spectrum management.
